Like Endpoints

  • GET /media/ media-id /likes Get a list of users who have liked this media.
  • POST /media/ media-id /likes Set a like on this media by the current user.
  • DEL /media/ media-id /likes Remove a like on this media by the current user.

Get a list of users who have liked this media.
The public_content scope is required for media that does not belong to the owner of the access_token .

Requirements
Scope: public_content
Parameters
access_token A valid access token.

POST /media/ media-id /likes

  • Response
curl -F 'access_token=ACCESS-TOKEN' \ https://api.instagram.com/v1/media/{media-id}/likes 
{
    "meta": {
        "code": 200
    }, 
    "data": null
}

Set a like on this media by the currently authenticated user.
The public_content scope is required for media that does not belong to the owner of the access_token .

Requirements
Scope: likes, public_content
Parameters
access_token A valid access token.

DEL /media/ media-id /likes

  • Response
curl -X DELETE https://api.instagram.com/v1/media/{media-id}/likes?access_token=ACCESS-TOKEN 
{
    "meta": {
        "code": 200
    }, 
    "data": null
}

Remove a like on this media by the currently authenticated user.
The public_content scope is required for media that does not belong to the owner of the access_token .

Requirements
Scope: likes, public_content
Parameters
access_token A valid access token.
