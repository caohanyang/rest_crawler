Error Messages

Product Advertising API returns error messages in English for the Amazon.com (US), Amazon.co.uk (UK), Amazon.de (DE), Amazon.fr (FR), and Amazon.ca (CA) locales.

Error messages are in Japanese for the Amazon.co.jp (JP) locale.

Error Code & Description Message Affected Operations

AWS.ExactParameterRequirement

This error indicates the length of your parameter is longer than allowed by the Product Advertising API.

Your request contains too much data for [ParameterName] . This parameter can have a maximum length of [MaximumNumber] . All

AWS.ExceededMaximumParameterValues

This error indicates the request has too many values for one or more parameters.

Your request contains too many values for [ParameterName] . This parameter can have a maximum of [MaximumNumber] values.

AWS.InsufficientParameterValues

This error indicates your request does not have the minimum number of values for a required parameter.

Your request contains too few values for [Parameter Name] . This parameter must have a minimum of [Minimum Value] values. All

AWS.InternalError

This error indicates an internal problem or outage is preventing Product Advertising API from completing your request.

For SOAP, this will be presented as a SOAP fault rather than an error.

We are unable to process your request at this time. Please retry your request. If you encounter this error repeatedly, please post a message on the AWS discussion board . All

AWS.InvalidAccount

This error indicates the request has an AWS access key ID in a locale where it is not registered for the Product Advertising API.

Your AccessKey Id is not registered for Product Advertising API. Please use the AccessKey Id obtained after registering at [RegistrationUrlForThatLocale] All

AWS.InvalidAssociate

This error indicates your account is not registered for Amazon Associates. If you see this error, verify:

  • You registered as an Amazon Associate in the requested locale. For more information, see Becoming an Associate .

  • You correctly added your Associate ID (tag) to your requests.

  • You added the email address registered for the Product Advertising API as a primary or secondary email address in your Amazon Associates account.

Your [AccessKeyId] is not registered as an Amazon Associate. Please register as an associate at [AssociateRegistrationUrlForThatLocale] . All

AWS.InvalidISO8601Time

This error indicates your request has a date or time value that is not formatted according to the ISO 8601 standard.

For example, this error is returned if your request has an invalid value for the Version parameter.

See Date and Time Formats .

[ParameterName] has an invalid value. It must contain a valid ISO 8601 date and time. All

AWS.InvalidOperationForMarketplace

This error indicates a request has an operation that is not supported for the locale.

This operation, [OperationName] , is not available for this locale. All

AWS.InvalidOperationParameter

This error indicates you chose an invalid operation.

For example, if your request has the AsinSearch operation, you will receive an error because AsinSearch is no longer supported.

The Operation parameter is invalid. Please modify the Operation parameter and retry. Valid values for the Operation parameter include [ListOfOperationValues] . All

AWS.InvalidParameterCombination

This error indicates two or more of the request parameters cannot be in the same request.

For example, if the CartAdd operation includes an ASIN and OfferListingId, you will receive an error.

Your request contains an invalid parameter combination. [ParameterName] and [ParameterName] cannot appear in the same request.

This error indicates the Product Advertising API does not recognize the shopping cart HMAC value in the request.

Your request contains an invalid value for HMAC. Please check your HMAC and retry your request. Remember that the HMAC must be URL-encoded if you are using REST.
