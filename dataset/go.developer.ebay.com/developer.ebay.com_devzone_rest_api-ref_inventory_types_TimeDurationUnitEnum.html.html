TimeDurationUnitEnum

This enumeration type is used by the fulfillmentTime field of the PickupAtLocationAvailability type to specify the time unit that is used to indicate the fulfillment time, which is how soon the item will be ready for In-Store Pickup or Click and Collect after the order has occurred.

Note: The TimeDurationUnitEnum type is currently not applicable, since this type is only used by the TimeDuration type, which is not yet available for use. In the future, the TimeDuration type will allow sellers to set the amount of time it will take for an In-Store Pickup or Click and Collect order to be ready at one or more of the merchant's physical stores.

Type that uses TimeDurationUnitEnum:

    TimeDuration

Calls that use one or more values of TimeDurationUnitEnum:

    /inventory_item GET , /inventory_item/{sku} PUT

Enumeration Values

Value Description
BUSINESS_DAY This enumeration value indicate that business days are used as the time unit for fulfillment time. This means that only business days, or days that the store is open, count towards the fullfillment time, and not days when the store is closed.
CALENDAR_DAY This enumeration value indicate that calendar days are used as the time unit for fulfillment time. This means that weekends and holidays are also included, and not just business days when the store is open.
DAY This enumeration value indicate that days are used as the time unit for fulfillment time.
HOUR This enumeration value indicate that hours are used as the time unit for fulfillment time.
MILLISECOND This enumeration value indicate that millliseconds are used as the time unit for fulfillment time.
MINUTE This enumeration value indicate that minutes are used as the time unit for fulfillment time.
MONTH This enumeration value indicate that months are used as the time unit for fulfillment time.
SECOND This enumeration value indicate that seconds are used as the time unit for fulfillment time.
YEAR This enumeration value indicate that years are used as the time unit for fulfillment time.
