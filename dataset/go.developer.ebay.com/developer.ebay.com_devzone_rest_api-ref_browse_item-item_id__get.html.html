Get Item

GET /item/{item_id}


This call retrieves the details of a specific item, such as description, price, and other information the buyer needs to make a purchasing decision.

Note: If you want the URL to the View Item page with the affiliate tracking ID returned in the itemSummaries.itemAffiliateWebUrl field, you must pass in the X-EBAY-C-ENDUSERCTX header in the request.

For example:
X-EBAY-C-ENDUSERCTX affiliateCampaignId= ePNCampaignId ,affiliateReferenceId= referenceId





Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/buy/browse/v1/item/{item_id}

URI parameters

Parameter Type Required? Meaning
item_id string Required eBay identifier of an item. This Id is returned in the Item.itemId field of the Search for Items call and in the itemId column of the Item Feed file.
For example: v1|150006654560|5546382


HTTP request headers

All requests made to eBay REST operations require you to provide the authorization HTTP header for authentication.
See HTTP request headers for details.



OAuth request scope

This request requires either a user access token or an application access token with the following scope:

https://api.ebay.com/oauth/api_scope

See Getting Access Tokens for more information.



Payload model

This call has no request payload.
