LocationResponse

This type is used by the base response payload for the Get Inventory Locations call.

Call that uses LocationResponse:

    /location GET

Fields

href ( string )
limit ( integer )
locations ( array of InventoryLocationResponse )
next ( string )
offset ( integer )
prev ( string )
total ( integer )


href ( string ) [0..?]
This is the URL to the current page of inventory locations.

Max length : 2048
See the Field Index to learn which calls use href.

limit ( integer ) [0..?]
This integer value is the the number of inventory locations that will be displayed on each results page.
See the Field Index to learn which calls use limit.

locations ( array of InventoryLocationResponse ) [0..?]
An array of one or more of the merchant's inventory locations.
See the Field Index to learn which calls use locations.

next ( string ) [0..?]
This is the URL to the next page of inventory locations. This field will only be returned if there are one or more additional pages of inventory locations to view.

Max length : 2048
See the Field Index to learn which calls use next.

offset ( integer ) [0..?]
This value indicates the page number of the result set. A value of 0 represents the first page of results, a value of 1 represents the second page of results, and so on.
See the Field Index to learn which calls use offset.

prev ( string ) [0..?]
This is the URL to the previous page of inventory locations. This field will only be returned if there are one or more previous pages of inventory locations to view.

Max length : 2048
See the Field Index to learn which calls use prev.

total ( integer ) [0..?]
Returns the total number of inventory locations in the paginated collection.
See the Field Index to learn which calls use total.
