Compatibility

This type is used by the Create or Replace Product Compatibility call to associate compatible vehicles to an inventory item. This type is also the base response of the Get Product Compatibility call.

Call that uses Compatibility:

    /inventory_item/{sku}/product_compatibility PUT

Fields

compatibleProducts ( array of CompatibleProduct ) [0..?]
This container consists of an array of motor vehicles (make, model, and year) that are compatible with the motor vehicle part or accessory specified by the sku value.
See the Field Index to learn which calls use compatibleProducts.

sku ( string ) [0..?]
This is the seller-defined SKU value of the inventory item that will be associated with the compatible vehicles. This field is not applicable to the Create or Replace Product Compatibility call, but it is always returned with the Get Product Compatibility call. For the the Create or Replace Product Compatibility call, the SKU value for the inventory item is actually passed in as part of the call URI.
See the Field Index to learn which calls use sku.
