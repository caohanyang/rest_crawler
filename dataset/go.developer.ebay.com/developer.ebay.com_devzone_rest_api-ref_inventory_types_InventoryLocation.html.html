InventoryLocation

This type is used by the Update Inventory Location call to update operating hours, special hours, phone number and other minor details of an inventory location.

Call that uses InventoryLocation:

    /location/{merchantLocationKey}/update_location_details POST

Fields

locationAdditionalInformation ( string )
locationInstructions ( string )
locationWebUrl ( string )
name ( string )
operatingHours ( array of OperatingHours )
phone ( string )
specialHours ( array of SpecialHours )


locationAdditionalInformation ( string ) [0..?]
This text field is used by the merchant to provide/update additional information about an inventory location.

Max length : 256
See the Field Index to learn which calls use locationAdditionalInformation.

locationInstructions ( string ) [0..?]
This text field is used by the merchant to provide/update special pickup instructions for the store location. Although this field is optional, it is recommended that merchants provide this field to create a pleasant and easy pickup experience for In-Store Pickup and Click and Collect orders. If this field is not included in the call payload, eBay will use the default pickup instructions contained in the merchant's profile.

Max length : 1000
See the Field Index to learn which calls use locationInstructions.

locationWebUrl ( string ) [0..?]
This text field is used by the merchant to provide/update the Website address (URL) for the store location.

Max length : 512
See the Field Index to learn which calls use locationWebUrl.

name ( string ) [0..?]
This text field is used by the merchant to update the name of the location. This is generally a human-friendly name as it is displayed in user interfaces.
See the Field Index to learn which calls use name.

operatingHours ( array of OperatingHours ) [0..?]
This container is used to provide/update the regular operating hours for a store location during the days of the week. A dayOfWeekEnum field and an intervals container will be needed for each day of the week that the store location is open. Note that if operating hours are already set for an inventory location for a specific day of the week, whatever is set through an Update Interval Location call will override those hours.
See the Field Index to learn which calls use operatingHours.

phone ( string ) [0..?]
This text field is used by the merchant to provide/update the phone number for the inventory location.

Max length : 36
See the Field Index to learn which calls use phone.

specialHours ( array of SpecialHours ) [0..?]
This container is used to provide/update the special operating hours for a store location on a specific date. The special hours specified for the specific date will override the normal operating hours for that particular day of the week. If special hours have already been set up for an inventory location, specifying special hours through an Update Interval Location call will only add to the list, unless the date(s) used are the same special date(s) already set up, in which case, the special hours set up through the Update Interval Location call will override the existing special hours.
See the Field Index to learn which calls use specialHours.
