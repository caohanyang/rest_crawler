Get Account Privileges

GET /privilege

Use this call to retrieve the seller's current set of privileges. The call returns whether or not the seller's eBay registration has been completed, as well as the details of their site-wide sellingLimt (the amount and quantity they can sell on a given day).

Response field descriptions



Output Container/Field Type Occurrence Meaning
sellerRegistrationCompleted boolean Conditionally Set this value to true to indicate the seller's registration is completed.
sellingLimit SellingLimit Conditionally Complex type defining the selling limits for the associated seller account.
sellingLimit. amount Amount Conditionally The maximum amount, in the seller's default currency, that an eBay seller can sell per calendar day.
sellingLimit.amount. currency string Conditionally The currency in which the value is expressed in. The currency is represented as a 3-letter ISO4217 currency code.

Applicable values are from CurrencyCodeEnum :See currency .
Code so that your app gracefully handles any future changes to this list.
sellingLimit.amount. value string Conditionally The value of the Amount in the specified currency.
sellingLimit. quantity integer Conditionally The maximum quantity of items that an eBay seller can sell per calendar day. Note that a listing with variations, all the items listed in the variation count as individual items.

Sample: Get Account Privileges

This call retrieves the account privileges for the seller.

Description

This call returns the privileges for a seller. Privileges include the seller's sellingLimit , quantity , and sellerRegistrationCompleted .

Input

This call does not use a request payload. 

URL format. See also the non-wrapped version of this URL.

GET https://api.sandbox.ebay.com/sell/account/v1/privilege

Output

If the call is successful, it returns a payload that contains the values of all the seller's privileges. 

JSON format.
{
  "sellingLimit": {
    "amount": {
      "value": "100.0",
      "currency": "USD"
    },
    "quantity": 10
  },
  "sellerRegistrationCompleted": true
}
