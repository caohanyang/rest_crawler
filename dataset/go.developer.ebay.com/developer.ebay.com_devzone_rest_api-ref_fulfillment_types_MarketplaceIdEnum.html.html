MarketplaceIdEnum

This enumerated type contains a list of the standard codes that represent each of the eBay marketplaces.

Type that uses MarketplaceIdEnum:

    LineItem

Calls that use one or more values of MarketplaceIdEnum:

    /order GET , /order/{orderId} GET
