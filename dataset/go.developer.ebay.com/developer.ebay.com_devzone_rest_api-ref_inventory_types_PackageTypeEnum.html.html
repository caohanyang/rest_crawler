PackageTypeEnum

This enumerated type is used to indicate what type of shipping package will be used to ship an inventory item.

Type that uses PackageTypeEnum:

    PackageWeightAndSize

Calls that use one or more values of PackageTypeEnum:

    /inventory_item GET , /inventory_item/{sku} PUT
