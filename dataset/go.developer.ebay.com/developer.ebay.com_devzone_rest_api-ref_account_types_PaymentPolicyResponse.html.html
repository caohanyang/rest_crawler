PaymentPolicyResponse

The response payload for payment policy requests.

Call that uses PaymentPolicyResponse:

    /payment_policy GET

Fields

href ( string )
limit ( integer )
next ( string )
offset ( integer )
paymentPolicies ( array of PaymentPolicy )
prev ( string )
total ( integer )


href ( string ) [0..?]
Returns a URL link to the current result set.
See the Field Index to learn which calls use href.

limit ( integer ) [0..?]
Returns the maximum number of results that can be returned in result set.
See the Field Index to learn which calls use limit.

next ( string ) [0..?]
Returns a URL link to the next set of results.
See the Field Index to learn which calls use next.

offset ( integer ) [0..?]
Returns how many result sets were skipped before the currently returned result set.
See the Field Index to learn which calls use offset.

paymentPolicies ( array of PaymentPolicy ) [0..?]
A list of the seller's payment policies.
See the Field Index to learn which calls use paymentPolicies.

prev ( string ) [0..?]
Returns a URL link to the previous set of results.
See the Field Index to learn which calls use prev.

total ( integer ) [0..?]
Returns the total number of result sets in the paginated collection.
See the Field Index to learn which calls use total.
