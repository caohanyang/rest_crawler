DiscountBenefit

Type defining the fields for a discount.

Refer to Item Promotion Rules for more information about using the discountRules and discountSpecificiation fields together to create various promotions.

Type that uses DiscountBenefit:

    DiscountRule

Calls that use DiscountBenefit:

    /item_promotion POST , /item_promotion/{promotion_id} GET

Fields

amountOffItem ( Amount ) [0..?]
The amount of discount that will be applied to an item when the promotion criteria is met. This would be used to create an 'Extra $5 off each item' offer. Every item the buyer purchases from the inventory set gets a $5 discount.

To create this promotion, you set the following discount details:
  • discountSpecification.minQuantity to 0
  • discountSpecification.forEachQuantity to 1
  • discountSpecification.numberOfDiscountedItems to 0
  • discountRules.discountBenefit.amountOffItem to 5


Note: The request can have only one of these fields: percentageOffOrder , amountOffOrder , percentageOffItem , or amountOffItem . Required: discountSpecification.numberOfDiscountedItems

Required unless the promotion is in DRAFT state.





See the Field Index to learn which calls use amountOffItem.

amountOffOrder ( Amount ) [0..?]
The total amount of the discount for an order.

Note: The request can have only one of these fields: percentageOffOrder , amountOffOrder , percentageOffItem , or amountOffItem and the request cannot have discountSpecification.numberOfDiscountedItems .

Required unless the promotion is in DRAFT state.



See the Field Index to learn which calls use amountOffOrder.

percentageOffItem ( string ) [0..?]
The percentage of discount that will be applied each time the promotion criteria is met. This would be used to create a 'Buy 1, get 1 at 30% off' promotion. If the buyer purchases 2 of the given item, they get a 30% discount off one of the items. If they purchase 4, they get a 30% discount off 2 of the items.

To create this promotion, you set the following discount details:
  • discountSpecification.minQuantity to 0
  • discountSpecification.forEachQuantity to 1
  • discountSpecification.numberOfDiscountedItems to 1
  • discountRules.discountBenefit.percentageOffItem to 30


Valid Values: 5, 10, 15, 20, 25, 30, 35, 40, 45, 50, 55, 60, 65, 70, 75, 80

Note: The request can have only one of these fields: percentageOffOrder , amountOffOrder , percentageOffItem , or amountOffItem .

Required: discountSpecification.numberOfDiscountedItems

Required unless the promotion is in DRAFT state.









See the Field Index to learn which calls use percentageOffItem.

percentageOffOrder ( string ) [0..?]
The total of the percentage discount for the order. This can be used to create an 'Extra 30% off' promotion. If the buyer purchases any items in the specified inventory set, then they get a 30% discount off those items.

To create this promotion, you set the following discount details:
  • discountSpecification.minQuantity to 1
  • discountSpecification.forEachQuantity to 0
  • discountSpecification.numberOfDiscountedItems to 0
  • discountRules.discountBenefit.percentageOffOrder to 30


Valid Values: 5, 10, 15, 20, 25, 30, 35, 40, 45, 50, 55, 60, 65, 70, 75, 80

Note: The request can have only one of these fields: percentageOffOrder , amountOffOrder , percentageOffItem , or amountOffItem .

Required unless the promotion is in DRAFT state.







See the Field Index to learn which calls use percentageOffOrder.
