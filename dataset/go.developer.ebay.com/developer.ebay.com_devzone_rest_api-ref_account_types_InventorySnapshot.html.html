InventorySnapshot

A view of an inventory at a specific time, or over a specified period.

No calls use InventorySnapshot.

Fields

cumulativeInventoryValue ( Amount )
cumulativeQuantity ( integer )
lookbackPeriod ( LookbackPeriod )
maxAvailableQuantity ( integer )
maxInventoryValue ( Amount )
minAvailableQuantity ( integer )
minInventoryValue ( Amount )


cumulativeInventoryValue ( Amount ) [0..?]
Not used by any call.

The cumulative value of an inventory over the specified lookback period.
See the Field Index to learn which calls use cumulativeInventoryValue.

cumulativeQuantity ( integer ) [0..?]
Not used by any call.

The cumulative quantity of items in the inventory over the specified lookback period.
See the Field Index to learn which calls use cumulativeQuantity.

lookbackPeriod ( LookbackPeriod ) [0..?]
Not used by any call.

The "lookback period" is the period of time to which the inventory snapshot applies.
See the Field Index to learn which calls use lookbackPeriod.

maxAvailableQuantity ( integer ) [0..?]
Not used by any call.

Maximum daily inventory quantity of items over the specified lookback period.
See the Field Index to learn which calls use maxAvailableQuantity.

Not used by any call.

Maximum daily inventory value over the specified lookback period.
See the Field Index to learn which calls use maxInventoryValue.

Not used by any call.

Minimum of daily inventory quantity of items over the specified lookback period.
See the Field Index to learn which calls use minAvailableQuantity.

Not used by any call.

Minimum of daily inventory value over the specified lookback period.
