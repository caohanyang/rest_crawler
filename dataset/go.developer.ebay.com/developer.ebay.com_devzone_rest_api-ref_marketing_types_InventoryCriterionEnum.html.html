InventoryCriterionEnum

Enumerated type that defines the values that indicate if the promotion is being created using rules or a list of inventory.

Type that uses InventoryCriterionEnum:

    InventoryCriterion

Calls that use one or more values of InventoryCriterionEnum:

    /item_promotion POST , /item_promotion/{promotion_id} GET

Enumeration Values

Value Description
INVALID Indicates the value submitted was invalid. Correct the request and submit the call again.
INVENTORY_BY_RULE Indicates the promotion inventory was selected based on rules and item aspects. Note: This is currently not supported.
INVENTORY_BY_VALUE Indicates the promotion inventory was selected by providing either a list of the seller's inventory reference IDs or the eBay listing IDs.
* See the Enumeration Index to see exact use of each enumeration value in the API.
