LineItem_0

Contains information about a line items in a purchase order.

Types that use LineItem_0:

    GuestPurchaseOrder , PurchaseOrder

Calls that use LineItem_0:

    /guest_purchase_order/{purchaseOrderId} GET , /purchase_order/{purchaseOrderId} GET

Fields

image ( Image )
itemId ( string )
lineItemId ( string )
lineItemPaymentStatus ( LineItemPaymentStatusEnum )
lineItemStatus ( LineItemStatusEnum )
netPrice ( Amount_0 )
quantity ( integer )
seller ( Seller )
shippingDetail ( ShippingDetail )
title ( string )


image ( Image ) [0..?]
The URL of the item's primary image.
See the Field Index to learn which calls use image.

itemId ( string ) [0..?]
The identifier of the item.
See the Field Index to learn which calls use itemId.

lineItemId ( string ) [0..?]
The identifier of this line item. This is created by the Checkout API's Initiate Checkout Session and Initiate Guest Checkout Session calls.
See the Field Index to learn which calls use lineItemId.

lineItemPaymentStatus ( LineItemPaymentStatusEnum ) [0..?]
Indicates the payment status of this line item.
See the Field Index to learn which calls use lineItemPaymentStatus.

lineItemStatus ( LineItemStatusEnum ) [0..?]
Indicates the state of this line item.
See the Field Index to learn which calls use lineItemStatus.

netPrice ( Amount_0 ) [0..?]
The total cost for the items in this line item taking into account the quantity and applying any seller item discounts, such as Buy 1 Get 1. Note : This does not include any shipping discounts, shipping costs, or seller adjustments.
See the Field Index to learn which calls use netPrice.

quantity ( integer ) [0..?]
The number of individual items in this line item.
See the Field Index to learn which calls use quantity.

seller ( Seller ) [0..?]
The seller's name and if available logo image of the seller offering this item.
See the Field Index to learn which calls use seller.

shippingDetail ( ShippingDetail ) [0..?]
Information about the shipping provider used for this line item.
See the Field Index to learn which calls use shippingDetail.

title ( string ) [0..?]
The seller created title of the item.
See the Field Index to learn which calls use title.
