SetReturnPolicyResponse

Complex type that that gets populated with a response containing a return policy.

Calls that use SetReturnPolicyResponse:

    /return_policy POST , /return_policy/{return_policy_id} PUT

categoryTypes ( array of CategoryType ) [0..?]
For return policies, this field always returns ALL_EXCLUDING_MOTORS_VEHICLES (returns on motor vehicles are not processed through eBay flows.)

Default : ALL_EXCLUDING_MOTORS_VEHICLES (for return policies only)

See the Field Index to learn which calls use categoryTypes.

description ( string ) [0..?]
An optional seller-defined description of the return policy.

Max length : 250
See the Field Index to learn which calls use description.

extendedHolidayReturnsOffered ( boolean ) [0..?]
If this value is set to true , it indicates the seller offers an Extended Holiday Returns policy for their listings.

IMPORTANT: Extended Holiday Returns is a seasonally available feature that is offered on some eBay marketplaces. To see if the feature is enabled in any given year, check the eBay Seller Center Returns on eBay page of before the holiday season begins.

See the Field Index to learn which calls use extendedHolidayReturnsOffered.

marketplaceId ( MarketplaceIdEnum ) [0..?]
The ID of the eBay marketplace to which this return policy applies. If this value is not specified, value defaults to the seller's eBay registration site.
See the Field Index to learn which calls use marketplaceId.

name ( string ) [0..?]
A user-defined name for this return policy. Names must be unique for policies assigned to the same marketplace.

Max length : 64

See the Field Index to learn which calls use name.

refundMethod ( RefundMethodEnum ) [0..?]
Indicates the method the seller uses to compensate the buyer for returned items. The return method specified applies only to remorse returns .
See the Field Index to learn which calls use refundMethod.

restockingFeePercentage ( string ) [0..?]
Optionally set by the seller, the percentage charged if the seller charges buyers a a restocking fee when items are returned. The total amount charged to the buyer is the cost of the item multiplied by the percentage indicated in this field.
See the Field Index to learn which calls use restockingFeePercentage.

returnInstructions ( string ) [0..?]
This optional free-form string field lets the seller provide a detailed explanation of the return policy.

Max length : 5000 characters
See the Field Index to learn which calls use returnInstructions.

returnMethod ( ReturnMethodEnum ) [0..?]
This field indicates the method in which the seller handles non-money back return requests for remorse returns . Sellers can specify they either exchange or replace items.
See the Field Index to learn which calls use returnMethod.

returnPeriod ( TimeDuration ) [0..?]
This value indicates the length of time the seller accepts returns, the duration of which starts when the buyer receives the item.
See the Field Index to learn which calls use returnPeriod.

returnPolicyId ( string ) [0..?]
A unique eBay-assigned ID for this policy.
See the Field Index to learn which calls use returnPolicyId.

returnsAccepted ( boolean ) [0..?]
If set to true , it indicates the seller accepts returns.
See the Field Index to learn which calls use returnsAccepted.

returnShippingCostPayer ( ReturnShippingCostPayerEnum ) [0..?]
The seller uses this value to specify who is responsible for paying the return shipping charges, either the buyer or the seller.

Note that the seller is always responsible for return shipping costs for SNAD-related issues.
See the Field Index to learn which calls use returnShippingCostPayer.

warnings ( array of ErrorDetailV3 ) [0..?]
A list of warnings related to request. This field normally returns empty, which indicates the request did not generate any warnings.
See the Field Index to learn which calls use warnings.
