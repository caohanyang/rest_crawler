AdResponse

Type that defines the fields for an ad response.

Type that uses AdResponse:

    BulkAdResponse

Calls that use AdResponse:

    /ad_campaign/{campaign_id}/bulk_create_ads_by_listing_id POST , /ad_campaign/{campaign_id}/bulk_update_ads_bid_by_listing_id POST

Fields

adId ( string )
errors ( array of ErrorDetailV3 )
href ( string )
listingId ( string )
statusCode ( integer )


adId ( string ) [0..?]
Identifier of an ad. This ID was generated when the ad was created.
See the Field Index to learn which calls use adId.

errors ( array of ErrorDetailV3 ) [0..?]
An array of errors associated with the listing IDs.
See the Field Index to learn which calls use errors.

href ( string ) [0..?]
URL of the ad, which is used to retrieve the ad.
See the Field Index to learn which calls use href.

listingId ( string ) [0..?]
The eBay listing identifier. This is generated when the listing was created.
See the Field Index to learn which calls use listingId.

statusCode ( integer ) [0..?]
Indicates if the process was successful.
See the Field Index to learn which calls use statusCode.
