StoreTypeEnum

An enumerated type defining the inventory location types.

Types that use StoreTypeEnum:

    InventoryLocationFull , InventoryLocationResponse

Calls that use one or more values of StoreTypeEnum:

    /location GET , /location/{merchantLocationKey} POST

Enumeration Values

Value Description
STORE This enumeration value indicates the inventory location is a merchant's store.
WAREHOUSE This enumeration value indicates the inventory location is a merchant's warehouse. If the location type is not specified when an inventory location is created, the location type value will default to WAREHOUSE .
* See the Enumeration Index to see exact use of each enumeration value in the API.
