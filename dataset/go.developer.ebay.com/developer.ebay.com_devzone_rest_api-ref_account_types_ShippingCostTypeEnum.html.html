ShippingCostTypeEnum

An enum defining the supported ways that shipping costs are calculated.

Type that uses ShippingCostTypeEnum:

    ShippingOption

Calls that use one or more values of ShippingCostTypeEnum:

    /fulfillment_policy POST , /fulfillment_policy/get_by_policy_name GET , /fulfillment_policy/{fulfillmentPolicyId} GET

Value Description
