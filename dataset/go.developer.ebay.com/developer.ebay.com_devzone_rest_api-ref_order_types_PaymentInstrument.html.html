PaymentInstrument

Type that defines the fields for the reference information about the payment method.

Type that uses PaymentInstrument:

    PurchaseOrder

Call that uses PaymentInstrument:

    /purchase_order/{purchaseOrderId} GET

Fields

brand ( PaymentMethodBrand ) [0..?]
The name of the credit card company (brand).
See the Field Index to learn which calls use brand.

paymentMethodType ( PaymentMethodTypeEnum ) [0..?]
Indicates the method of payment, such as credit card.
See the Field Index to learn which calls use paymentMethodType.
