ItemReturnTerms

Type that defines the fields for the seller's return policy.

Type that uses ItemReturnTerms:

    Item

Call that uses ItemReturnTerms:

    /item/{item_id} GET

Fields

extendedHolidayReturnsOffered ( boolean ) [0..?]
This boolean field is returned as true if the seller has enabled the Extended Holiday Returns feature on the item. Extended Holiday Returns are only applicable during the US holiday season, and gives buyers extra time to return an item. This 'extra time' will typically extend beyond what is set through the returnPeriod value.
See the Field Index to learn which calls use extendedHolidayReturnsOffered.

refundMethod ( RefundMethodEnum ) [0..?]
This enumeration value indicates how a buyer is refunded when an item is returned.
See the Field Index to learn which calls use refundMethod.

restockingFeePercentage ( string ) [0..?]
This string field indicates the restocking fee percentage that the seller has set on the item. Sellers have the option of setting no restocking fee for an item, or they can set the percentage to 10, 15, or 20 percent. So, if the cost of the item was $100, and the restocking percentage was 20 percent, the buyer would be charged $20 to return that item, so instead of receiving a $100 refund, they would received $80 due to the restocking fee.
See the Field Index to learn which calls use restockingFeePercentage.

returnInstructions ( string ) [0..?]
Text written by the seller describing what the buyer should to do in order to return the item.
See the Field Index to learn which calls use returnInstructions.

returnMethod ( ReturnMethodEnum ) [0..?]
This field is returned if the seller is able to offer the buyer an item replacement or exchange instead of a monetary refund.
See the Field Index to learn which calls use returnMethod.

returnPeriod ( TimeDuration ) [0..?]
The amount of time the buyer has to return the item after the purchase date.
See the Field Index to learn which calls use returnPeriod.

returnsAccepted ( boolean ) [0..?]
Indicates whether or not the seller accepts returns for the item.
See the Field Index to learn which calls use returnsAccepted.

returnShippingCostPayer ( ReturnShippingCostPayerEnum ) [0..?]
This enumeration value indicates whether the buyer or seller is responsible for return shipping costs when an item is returned.
See the Field Index to learn which calls use returnShippingCostPayer.
