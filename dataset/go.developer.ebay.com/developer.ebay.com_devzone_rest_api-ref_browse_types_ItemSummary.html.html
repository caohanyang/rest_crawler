ItemSummary

This type defines the item details for a specific item.

Type that uses ItemSummary:

    SearchPagedCollection

Call that uses ItemSummary:

    /item_summary/search GET

Fields

bidCount ( integer )

additionalImages ( array of Image ) [0..?]
This container is an array of URLs for Gallery Plus images of the item listing.
See the Field Index to learn which calls use additionalImages.

bidCount ( integer ) [0..?]
This integer value indicates the total number of bids that have been placed for an auction item. This field is only returned for auction listings.
See the Field Index to learn which calls use bidCount.

buyingOptions ( array of string ) [0..?]
The value returned in this field indicates the buying option(s) for the item. FIXED_PRICE is returned for basic fixed-price listings, AUCTION is returned for auction listings without Buy It Now feature, and both FIXED_PRICE and AUCTION are returned for auction listings enabled with the Buy It Now feature.
See the Field Index to learn which calls use buyingOptions.

categories ( array of Category ) [0..?]
This container consists of the primary listing category ID of the item (as well as secondary listing category if item was listed in two categories).
See the Field Index to learn which calls use categories.

condition ( string ) [0..?]
The value returned in this field indicates the condition of the item.
See the Field Index to learn which calls use condition.

currentBidPrice ( Amount_0 ) [0..?]
This container displays the current highest bid for an auction item. The value field shows the dollar value of the current highest bid, and the currency field (3-digit ISO code) denotes the currency associated with that bid value. This field is only returned for auction listings.
See the Field Index to learn which calls use currentBidPrice.

distanceFromPickupLocation ( TargetLocation ) [0..?]
This container indicates the distance away that the item is from the the pickupPostalCode value that was supplied in the call request. This container is only returned if the 'local pickup' filter fields are used in the request.
See the Field Index to learn which calls use distanceFromPickupLocation.

energyEfficiencyClass ( string ) [0..?]
The value returned in this field indicates the energy efficiency rating of the item. Energy efficiency ratings apply to products listed by commercial vendors in electronics categories only. Currently, this field is only applicable for the Germany site, and this field is only returned if the seller specified the energy efficiency rating through item specifics at listing time. Rating values include A+++ , A++ , A+ , A , B , C , D , E , F , and G .
See the Field Index to learn which calls use energyEfficiencyClass.

image ( Image ) [0..?]
The URL to the primary image of the item.
See the Field Index to learn which calls use image.

itemAffiliateWebUrl ( string ) [0..?]
This URL is the link to the View Item page with an affiliate tracking ID included. This field is only returned if the seller enabled affiliate tracking for the listing. This field is only returned if the seller enables affiliate tracking for the listing. To enable affiliate tracking, the X-EBAY-C-ENDUSERCTX request header is required.
See the Field Index to learn which calls use itemAffiliateWebUrl.

itemGroupHref ( string ) [0..?]
A reference (in HATEOAS style) you use to retrieve a group of items with varying aspects.

For example : /buy/v1/item_group/15006693 .

See the Field Index to learn which calls use itemGroupHref.

itemId ( string ) [0..?]
The unique identifier of the item.

For example: v1|150006693022|450002439078

See the Field Index to learn which calls use itemId.

itemLocation ( ItemLocationImpl ) [0..?]
This container gives the location of the item. This container consists of fields you typically see for an address, including postal code, county, state/province, street address, city, and country (2-digit ISO code).
See the Field Index to learn which calls use itemLocation.

marketingPrice ( MarketingPrice_0 ) [0..?]
This container is returned if the item is eligible for a seller discount, and consists of the item's original price, and the seller discount amount and percentage.
See the Field Index to learn which calls use marketingPrice.

pickupOptions ( array of PickupOptionSummary ) [0..?]
This container lists the local pickup options available to the buyer. This container is only returned if the user is searching for local pickup items and set the local pickup filters in the call request.
See the Field Index to learn which calls use pickupOptions.

price ( Amount_0 ) [0..?]
The price of the item.
See the Field Index to learn which calls use price.

seller ( Seller ) [0..?]
This container consists of information about the seller of the item.
See the Field Index to learn which calls use seller.

shippingOptions ( array of ShippingOptionSummary ) [0..?]
This container lists the shipping options available to ship the item.
See the Field Index to learn which calls use shippingOptions.

thumbnailImages ( array of Image ) [0..?]
This container is an array of thumbnail images for the item. For each image, the URL to the image is given.
See the Field Index to learn which calls use thumbnailImages.

title ( string ) [0..?]
The seller-created title of the item.
See the Field Index to learn which calls use title.

topRatedBuyingExperience ( boolean ) [0..?]
This boolean field is returned as true for top-rated listings. As a buyer, three things that you get from a top-rated listing is a 30-day or longer return policy, same-day or 1-day handling time, and the added comfort of knowing that you are purchasing an item from a top-rated seller. See the Becoming a Top Rated Seller and qualifying for Top Rated Plus help topic for more information about Top Rated Seller and Top Rated Plus listing requirements.
See the Field Index to learn which calls use topRatedBuyingExperience.
