ProgramTypeEnum

An enum defining the supported seller programs.

Type that uses ProgramTypeEnum:

    Program

Calls that use one or more values of ProgramTypeEnum:

    /program/get_opted_in_programs GET , /program/opt_in POST , /program/opt_out POST

Enumeration Values

Value Description
OUT_OF_STOCK_CONTROL The Out of Stock Control program that sellers can participate in by opting-in to the program.
SELLING_POLICY_MANAGEMENT The selling policy management program that sellers can participate in by opting-in to the program. Sellers must opt-in to this program if they want eBay to associste the business policies they create with the items they list.
* See the Enumeration Index to see exact use of each enumeration value in the API.
