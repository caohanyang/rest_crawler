AdReference

Type that defines the fields for the ad ID and its URL.

Type that uses AdReference:

    AdReferences

Call that uses AdReference:

    /ad_campaign/{campaign_id}/create_ads_by_inventory_reference POST

Fields

adId ( string ) [0..?]
Identifier of an ad. This ID was generated when the ad was created.
See the Field Index to learn which calls use adId.

href ( string ) [0..?]
URL of the ad, which is used to retrieve the ad.
See the Field Index to learn which calls use href.
