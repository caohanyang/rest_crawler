Get Inventory Locations

GET /location

Use this call to retrieve the complete list of inventory locations for a seller. This call does not use a payload, although you can filter the returned locations using the limit and offset query parameters.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/inventory/v1/location?
  limit=integer&
  offset=integer

URI parameters

Parameter Type Required? Meaning
limit integer Optional The value passed in this query parameter sets the maximum number of records to return per page of data. The value passed in this field should be an integer from 1 to 100 . If this query parameter is not set, up to 100 records will be returned on each page of results.

Min : 1
offset integer Optional The value passed in this query parameter sets the page number to retrieve. The first page of records has a value of 0 , the second page of records has a value of 1 , and so on. If this query parameter is not set, its value defaults to 0 , and the first page of records is returned.

Sample: Get all Locations

This call retrieves all the locations that a seller has configured.

Description

Sellers can create one or more inventory locations. This call returns all the locations that the seller has set up.

Input

This call does not use a request payload. The user has the option of using the limit and offset to filter/control the inventory locations that are displayed. In this particular sample, all inventory locations will be retrieved with one call. 

URL format. See also the non-wrapped version of this URL.

GET https://api.ebay.com/sell/inventory/v1/location
