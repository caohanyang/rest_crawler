Get Opted-in Programs

GET /program/get_opted_in_programs

Use this call to get a list of the seller programs that the seller has opted-in to.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/account/v1/program/get_opted_in_programs

This call has no path or query parameters.


HTTP request headers

All requests made to eBay REST operations require you to provide the authorization HTTP header for authentication.

OAuth request scope

https://api.ebay.com/oauth/api_scope/sell.account.readonly

Response field descriptions



Output Container/Field Type Occurrence Meaning
programs array of Program Conditionally A list of seller programs.
programs. programType string Conditionally Program type.

Applicable values are from ProgramTypeEnum :

OUT_OF_STOCK_CONTROL
The Out of Stock Control program that sellers can participate in by opting-in to the program.
SELLING_POLICY_MANAGEMENT
The selling policy management program that sellers can participate in by opting-in to the program. Sellers must opt-in to this program if they want eBay to associste the business policies they create with the items they list.

Code so that your app gracefully handles any future changes to this list.

Sample: Get opted-in programs.

Retrieves the eBay seller programs to which the seller has enrolled.

Description

Sellers can opt-in to the seller programs offered by eBay. This call returns the programs in which the seller is participating.

Input

This call does not use a request payload. 

URL format. See also the non-wrapped version of this URL.

GET https://api.sandbox.ebay.com/sell/account/v1/program/get_opted_in_programs

Output

If the call is successful, eBay responds with the list of currently opted-in programs. 

JSON format.
{
  "programs": [
    {
      "programType": "OUT_OF_STOCK_CONTROL"
    },
    {
      "programType": "SELLING_POLICY_MANAGEMENT"
    }
  ]
}
