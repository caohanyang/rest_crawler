FindSellerStandardsProfilesResponse

The response container for a list of seller profiles.

Call that uses FindSellerStandardsProfilesResponse:

    /seller_standards_profile GET

Field

standardsProfiles ( array of StandardsProfile ) [0..?]
A list standards profiles for the seller. A standardsProfile is a combination of cycle and a program.
See the Field Index to learn which calls use standardsProfiles.
