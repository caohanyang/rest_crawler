LineItemReference

Type the defines the fields for the line item Id and the quantity.

Type that uses LineItemReference:

    ShippingFulfillment

Call that uses LineItemReference:

    /purchase_order/{purchaseOrderId} GET

Fields

lineItemId ( string ) [0..?]
A unique eBay-assigned Id value to identify the line item in a purchase order.
See the Field Index to learn which calls use lineItemId.

quantity ( integer ) [0..?]
The number of individual items ordered for this line item, as specified by the buyer.
See the Field Index to learn which calls use quantity.
