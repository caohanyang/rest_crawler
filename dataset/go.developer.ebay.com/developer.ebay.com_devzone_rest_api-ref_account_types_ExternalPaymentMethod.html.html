ExternalPaymentMethod

A container that describes a payment method.

No calls use ExternalPaymentMethod.

Fields

paymentMethodName ( string ) [0..?]
Not used by any call.

The name of the payment method if paymentMethodType equals OTHERS .
See the Field Index to learn which calls use paymentMethodName.

paymentMethodType ( ExternalPaymentMethodTypeEnum ) [0..?]
Not used by any call.

The payment method (such as CREDIT_CARD).
See the Field Index to learn which calls use paymentMethodType.

primary ( boolean ) [0..?]
Not used by any call.

Set this value to true to indicate this is the primary payment method.
See the Field Index to learn which calls use primary.

verified ( boolean ) [0..?]
Not used by any call.

Set this value to true to indicate this payment method has been verified.
See the Field Index to learn which calls use verified.
