CheckoutSessionResponse

Type the defines the fields for a checkout session.

Calls that use CheckoutSessionResponse:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId} GET , /checkout_session/{checkoutSessionId}/update_payment_info POST , /checkout_session/{checkoutSessionId}/update_quantity POST , /checkout_session/{checkoutSessionId}/update_shipping_address POST , /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId} GET , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_address POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST

Fields

acceptedPaymentMethods ( array of PaymentMethod )
checkoutSessionId ( string )

acceptedPaymentMethods ( array of PaymentMethod ) [0..?]
Type that defines the fields for the payment method that can be used.
See the Field Index to learn which calls use acceptedPaymentMethods.

checkoutSessionId ( string ) [0..?]
A unique eBay-assigned Id value the identifies a checkout session. For example,

Guest checkout session Id : v1|5688999|gh677u
eBay Member checkout session Id : 67667512387


See the Field Index to learn which calls use checkoutSessionId.

expirationDate ( string ) [0..?]
The expiration time for the checkout session.
See the Field Index to learn which calls use expirationDate.

lineItems ( array of LineItem ) [0..?]
An array of line items associated with the checkout session.
See the Field Index to learn which calls use lineItems.

pricingSummary ( PricingSummary ) [0..?]
Container for the details of the grand total of all the line items.
See the Field Index to learn which calls use pricingSummary.

providedPaymentInstrument ( ProvidedPaymentInstrument ) [0..?]
Container for the payment methods that can be used for the checkout. This is returned only if you have used the Update Payment Info call to change the payment method.
See the Field Index to learn which calls use providedPaymentInstrument.

shippingAddress ( ShippingAddress ) [0..?]
Container for the address to which the purchase order will be shipped. Note: If the address cannot be validated, a warning message will be returned.
See the Field Index to learn which calls use shippingAddress.

warnings ( array of ErrorDetailV3 ) [0..?]
An array of any process errors or warnings that were generated during the call processing.
See the Field Index to learn which calls use warnings.
