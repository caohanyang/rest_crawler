Offers

This type is used by the base response of the Get Offers call, and it is an array of one or more of the seller's offers, along with pagination data.

Call that uses Offers:

    /offer GET

Fields

href ( string )
limit ( integer )
next ( string )
offers ( array of EbayOfferDetailsWithAll )
prev ( string )
size ( integer )
total ( integer )


href ( string ) [0..?]
This is the URL to the current page of offers.
See the Field Index to learn which calls use href.

limit ( integer ) [0..?]
This integer value is the the number of offers that will be displayed on each results page.
See the Field Index to learn which calls use limit.

next ( string ) [0..?]
This is the URL to the next page of offers. This field will only be returned if there are additional offers to view.
See the Field Index to learn which calls use next.

offers ( array of EbayOfferDetailsWithAll ) [0..?]
This container is an array of one or more of the seller's offers. With the Get Offers call, the seller has the options of retrieving: all current offers for all products on all eBay marketplaces (don't use any query parameters); all current offers for a specific product on all eBay marketplaces (use the sku query parameter); all current offers for all products on a specific eBay marketplace (use the marketplaceId query parameter); or a current offer for a specific product on a specific eBay marketplace (use the sku and marketplaceId query parameters).
See the Field Index to learn which calls use offers.

prev ( string ) [0..?]
This is the URL to the previous page of offers. This field will only be returned if there are previous offers to view.
See the Field Index to learn which calls use prev.

size ( integer ) [0..?]
This integer value indicates the total number of pages of results that are available. This number will depend on the total number of offers available for viewing, and on the limit value.
See the Field Index to learn which calls use size.

total ( integer ) [0..?]
This integer value is the total number of offers that exist for the seller's account. Based on this number and on the limit value, the seller may have to toggle through multiple pages to view all offers
See the Field Index to learn which calls use total.
