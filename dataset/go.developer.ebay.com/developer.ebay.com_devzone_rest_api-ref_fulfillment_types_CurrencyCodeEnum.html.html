CurrencyCodeEnum

This enumerated type lists the three-letter ISO 4217 code representing the supported world currencies.

Type that uses CurrencyCodeEnum:

    Amount

Calls that use one or more values of CurrencyCodeEnum:

    /order GET , /order/{orderId} GET
