ShippingAddress_0

Type that defines the fields for a shipping address.

Type that uses ShippingAddress_0:

    PurchaseOrder

Call that uses ShippingAddress_0:

    /purchase_order/{purchaseOrderId} GET

Fields

addressLine1 ( string )
addressLine2 ( string )
city ( string )
country ( CountryCodeEnum )
county ( string )
phoneNumber ( string )
postalCode ( string )
recipient ( string )
stateOrProvince ( string )


addressLine1 ( string ) [0..?]
The first line of the street address where the item is being shipped.
See the Field Index to learn which calls use addressLine1.

addressLine2 ( string ) [0..?]
The second line of the street address where the item is being shipped. This optional field can be used for information such as 'Suite Number' or 'Apt Number'.
See the Field Index to learn which calls use addressLine2.

city ( string ) [0..?]
The city of the address where the item is being shipped.
See the Field Index to learn which calls use city.

country ( CountryCodeEnum ) [0..?]
The two letter code representing the country of the address.
See the Field Index to learn which calls use country.

county ( string ) [0..?]
The county of the address where the item is being shipped.
See the Field Index to learn which calls use county.

phoneNumber ( string ) [0..?]
The phone number of the person receiving the package.
See the Field Index to learn which calls use phoneNumber.

postalCode ( string ) [0..?]
The postal code of the address where the item is being shipped.
See the Field Index to learn which calls use postalCode.

recipient ( string ) [0..?]
Name of the person receiving the package.
See the Field Index to learn which calls use recipient.

stateOrProvince ( string ) [0..?]
The state or province of the address where the item is being shipped.
See the Field Index to learn which calls use stateOrProvince.
