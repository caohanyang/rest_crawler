PriceDisplayConditionEnum

Enumeration type that defines the values that specify where/when the price of an item is shown.

Types that use PriceDisplayConditionEnum:

    Item , ItemDigest

Calls that use one or more values of PriceDisplayConditionEnum:

    /item/{item_id} GET , /item_group/{item_group_id} GET

Enumeration Values

Value Description
ALWAYS_SHOW Indicates there are no restrictions as to when/where the price of the item can be shown.
ONLY_SHOW_WHEN_ADDED_IN_CART Indicates the price of the item can be shown only after it has been added to the cart.
