Delete Offer

DELETE /offer/{offerId}

This call deletes a specific published or unpublished offer. In the case of a published offer (or live eBay listing), a successful call will either end the single-variation listing associated with the offer, or it will remove the product variation associated with that offer in a multiple-variation offer.
