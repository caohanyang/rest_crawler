SetFulfillmentPolicyResponse

Complex type that that gets populated with a response containing a fulfillment policy.

Calls that use SetFulfillmentPolicyResponse:

    /fulfillment_policy POST , /fulfillment_policy/{fulfillmentPolicyId} PUT

Fields

categoryTypes ( array of CategoryType )
description ( string )
freightShipping ( boolean )
fulfillmentPolicyId ( string )
globalShipping ( boolean )
handlingTime ( TimeDuration )
localPickup ( boolean )
marketplaceId ( MarketplaceIdEnum )
name ( string )
pickupDropOff ( boolean )
shippingOptions ( array of ShippingOption )
shipToLocations ( RegionSet )
warnings ( array of ErrorDetailV3 )


categoryTypes ( array of CategoryType ) [0..?]
The CategoryTypeEnum value to which this policy applies. Used to discern accounts that sell motor vehicles from those that don't. (Currently, each policy can be set to only one categoryTypes value at a time.)
See the Field Index to learn which calls use categoryTypes.

description ( string ) [0..?]
An optional seller-defined description of the fulfillment policy.

Max length : 250
See the Field Index to learn which calls use description.

freightShipping ( boolean ) [0..?]
If this value is set to true , it indicates the seller offers freight shipping.
See the Field Index to learn which calls use freightShipping.

fulfillmentPolicyId ( string ) [0..?]
A unique eBay-assigned ID for this policy.
See the Field Index to learn which calls use fulfillmentPolicyId.

globalShipping ( boolean ) [0..?]
If this value is set to true , it indicates the Global Shipping Program is the default international shipping option for the seller, and eBay sets the international shipping service to International Priority Shipping.
See the Field Index to learn which calls use globalShipping.

handlingTime ( TimeDuration ) [0..?]
Specifies the maximum number of business days the seller commits to for preparing and shipping an order after receiving a cleared payment for the order. This time does not include the transit time it takes the shipping carrier to deliver the order.

Note that Top-Rated sellers must offer same-day or one-day handling for a listing in order for that listing to receive a Top Rated Plus seal on the View Item or Search Results pages.

Min : 0 Max : 30



See the Field Index to learn which calls use handlingTime.

localPickup ( boolean ) [0..?]
If this value is set to true , it indicates the seller offers local pickup of their items.
See the Field Index to learn which calls use localPickup.

marketplaceId ( MarketplaceIdEnum ) [0..?]
The ID of the eBay marketplace to which this fulfillment policy applies. If this value is not specified, value defaults to the seller's eBay registration site.
See the Field Index to learn which calls use marketplaceId.

name ( string ) [0..?]
A user-defined name for this fulfillment policy. Names must be unique for policies assigned to the same marketplace.

Max length : 64

See the Field Index to learn which calls use name.

pickupDropOff ( boolean ) [0..?]
If this value is set to true , it indicates the seller offers the "Click and Collect" feature to be used with Add/Revise/Relist calls.

To enable "Click and Collect", a seller must (1) be eligible for the "Click and Collect" feature, and (2) set this boolean field to 'true'. Currently, the "Click and Collect" feature is available only to large retail merchants the eBay AU and UK marketplaces.
See the Field Index to learn which calls use pickupDropOff.

shippingOptions ( array of ShippingOption ) [0..?]
A list of default shipping options supported by the seller. Shipping options can include the type of shipping, the shipping and handling costs, and any applicable insurance information. Use the optionType field to set up either domestic or international shipping options.
See the Field Index to learn which calls use shippingOptions.

shipToLocations ( RegionSet ) [0..?]
The geographical region(s) to which the seller ships.
It is not necessary to include the shipToLocations field; eBay automatically adds the seller's listing country as a 'Ship-To Location', and the field is always returned in the shipping policy response.
If the seller wants to offer international shipping as part of the shipping policy, supply at least one region in shipToLocation field that is outside the seller's listing country. To offer shipping to every region and country (supported by eBay shipping services), the seller can pass in 'WORLDWIDE' as a shipToLocation value.

See the Field Index to learn which calls use shipToLocations.

warnings ( array of ErrorDetailV3 ) [0..?]
A list of warnings related to request. This field normally returns empty, which indicates the request did not generate any warnings.
See the Field Index to learn which calls use warnings.
