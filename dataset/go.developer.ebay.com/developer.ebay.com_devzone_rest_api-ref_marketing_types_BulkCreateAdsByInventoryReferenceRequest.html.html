BulkCreateAdsByInventoryReferenceRequest

Type that defines the fields used to create ads using the seller's inventory references in bulk.

Calls that use BulkCreateAdsByInventoryReferenceRequest:

    /ad_campaign/{campaign_id}/bulk_create_ads_by_inventory_reference POST , /ad_campaign/{campaign_id}/bulk_update_ads_bid_by_inventory_reference POST

Field

requests ( array of CreateAdsByInventoryReferenceRequest ) [0..?]
An array of the seller's inventory reference/inventory type pairs and the bid percentage for each to be used to create ads.
See the Field Index to learn which calls use requests.
