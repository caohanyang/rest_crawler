MarketplaceIdEnum

Enumeration value representing an eBay site.

Types that use MarketplaceIdEnum:

    Campaign , CreateCampaignRequest , ItemPromotion , ItemPromotionResponse , PromotionDetail

Calls that use one or more values of MarketplaceIdEnum:

    /ad_campaign POST , /ad_campaign/find_campaign_by_ad_reference GET , /ad_campaign/get_campaign_by_name GET , /ad_campaign/{campaign_id} GET , /item_promotion POST , /item_promotion/{promotion_id} GET , /promotion GET
