Campaign

Type that defines the fields for a campaign.

Types that use Campaign:

    CampaignPagedCollection , Campaigns

Calls that use Campaign:

    /ad_campaign GET , /ad_campaign/find_campaign_by_ad_reference GET , /ad_campaign/get_campaign_by_name GET , /ad_campaign/{campaign_id} GET

Fields

campaignCriterion ( CampaignCriterion )
campaignId ( string )
campaignName ( string )
campaignStatus ( CampaignStatusEnum )
endDate ( string )
fundingStrategy ( FundingStrategy )
marketplaceId ( MarketplaceIdEnum )
startDate ( string )


campaignCriterion ( CampaignCriterion ) [0..?]
The selection rules (criterion) used to select the listings for a campaign.
See the Field Index to learn which calls use campaignCriterion.

campaignId ( string ) [0..?]
The unique identifier of the campaign. This ID was generated when the campaign was created.
See the Field Index to learn which calls use campaignId.

campaignName ( string ) [0..?]
The name of the campaign.
See the Field Index to learn which calls use campaignName.

campaignStatus ( CampaignStatusEnum ) [0..?]
Indicates the state of the campaign.
See the Field Index to learn which calls use campaignStatus.

endDate ( string ) [0..?]
The end date of the campaign.

Valid Format: yyyy-MM-ddThh:mm.ss.sssZ

See the Field Index to learn which calls use endDate.

fundingStrategy ( FundingStrategy ) [0..?]
The container for the funding strategy (bid percentage and funding model) of the campaign.
See the Field Index to learn which calls use fundingStrategy.

marketplaceId ( MarketplaceIdEnum ) [0..?]
The marketplace ID in which the campaign is associated.
See the Field Index to learn which calls use marketplaceId.

startDate ( string ) [0..?]
The start date of the campaign. On this day, the service derives the keywords for each listing, creates an ad for each listing as associates the ad with the campaign and then starts the campaign. The amount of time it takes the service to start the campaign depends on the number of listings in the campaign. You can check it the campaign has started by using the Get Campaign call.

Valid Format: yyyy-MM-ddThh:mm.ss.sssZ

See the Field Index to learn which calls use startDate.
