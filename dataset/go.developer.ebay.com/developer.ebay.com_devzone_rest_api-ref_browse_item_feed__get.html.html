Get Item Feed

GET /item_feed


This call returns the items in the specified feed file. The first line is the Header , which indicates the order of the values on each line. Each feed file is for a specific category and date. The category names and IDs are for the US site only. Most of these category IDs will work on other sites, but not all. All fields are always in the file but may be empty.

Note: Feed files are generated only for items on the US eBay site. All items in this file are:

  • From Top Rated eBay Sellers
  • Have been added in the last 24 hours as of the date of the feed file
  • Are fixed price; not auctions (Buy It Now)

Note: In order to use this call, you must join the eBay Partner Network . For information about the program see detail about the eBay Partner Network .

Resource URI (production) 

GET https://api.ebay.com/buy/browse/v1/item_feed?
  category_id=string&
  date=string&
  feed_type=string

URI parameters

Parameter Type Required? Meaning
category_id string Required The eBay category identifier of the items to be returned in the feed file. For a list of the categories that are supported, see Item Feed API categories .
date string Required The date of the feed you want returned. This can be up to 6 days in the past but cannot be set to a date in the future.

Note: Generating the feed files each day takes time. If you request a file for the current date and get an error saying it doesn't exist, just try the call again later in the day. Format: yyyymmdd
Requirement: Must be within the last 6 days


feed_type string Required The eBay Partners Network feed type. The only value supported is DAILY .

Response field descriptions



Output Container/Field Type Occurrence Meaning
itemFeed array of ItemFeed Always An array of eBay items.
itemFeed. additionalImageUrls array of string Always An array of item image URLs that does not include the primary image for the item. The URL for the primary image is returned in the imageUrl field.
itemFeed. ageGroup string Always The age group for which the product is recommended. For example, newborn, infant, toddler, kids, adult, etc.
itemFeed. brand string Always The name brand of the item, such as Nike, Apple, etc.
itemFeed. category string Always The name of the eBay category associated with the item.
itemFeed. categoryId string Always The identifier of the leaf category associated with the item.
itemFeed. color string Always The text describing the color of the item.
itemFeed. condition string Always The text describing the condition of the item.
itemFeed. conditionId string Always The identifier of the condition of the item.
itemFeed. description string Always Text describing the item.
itemFeed. gender string Always The gender for the item. This is used for items that could vary by gender, such as clothing. For example: male, female, or unisex.
itemFeed. gtin string Always The unique Global Trade Item, (UPC, ISBN, EAN) number of the item as defined by http://www.gtin.info .
itemFeed. imageUrl string Always The URL to the primary image of the item. The other images of the item are returned in the additionalImageUrls field.
itemFeed. itemEndDate datetime Always A timestamp indicating when the item's sale period will end based on its start date and duration. The format is yyyy-mm-dd.
itemFeed. itemId string Always The unique identifier of the eBay item.
itemFeed. material string Always The text describing what the item is made of. For example, silk or plastic.
itemFeed. mpn string Always The manufacturer part number, which is a unique number that identifies a specific product.
itemFeed. priceCurrency string Always The currency used for the price of the item. Generally, this is the currency used by the country of eBay site offering the item.

Applicable values are from CurrencyCodeEnum :See priceCurrency .

itemFeed. priceValue string Always The dollar amount of the item specified in the currency shown in the priceCurrency field.
itemFeed. quantity integer Always A number representing how many of this item are available for purchase.
