Pause a Promotion

POST /promotion/{promotion_id}/pause


This call pauses a promotion and changes the start from RUNNING to PAUSED. Only running promotions can be paused. To use the call, you pass in the promotion Id as a URI parameter.

This is used to temporarily make a promotion unavailable. Any incomplete transactions would not receive the offer until the promotion was resumed. No offer teasers are visible to users when a promotion is paused.

You can use the Get Promotions call retrieve the IDs of the seller's promotions.

Resource URI (production) 

POST https://api.ebay.com/sell/marketing/v1/promotion/{promotion_id}/pause

URI parameters

Parameter Type Required? Meaning
promotion_id string Required Identifier of the promotion. This Id was generated when the promotion was created. You can use the Get Promotions call retrieve the IDs of the seller's promotions.
