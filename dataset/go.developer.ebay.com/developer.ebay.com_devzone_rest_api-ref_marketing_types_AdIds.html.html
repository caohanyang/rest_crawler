AdIds

Type that defines the container for an array of ad IDs.

Call that uses AdIds:

    /ad_campaign/{campaign_id}/delete_ads_by_inventory_reference POST

Field

adIds ( array of string ) [0..?]
An array of the ID for the ads that were created.
