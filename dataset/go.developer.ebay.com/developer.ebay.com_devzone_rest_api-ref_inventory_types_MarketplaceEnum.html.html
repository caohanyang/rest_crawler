MarketplaceEnum

This enumeration type is used to indicate the different eBay marketplace sites.

Types that use MarketplaceEnum:

    EbayOfferDetailsWithAll , EbayOfferDetailsWithKeys , FeeSummary

Calls that use one or more values of MarketplaceEnum:

    /offer POST , /offer/get_listing_fees POST , /offer/{offerId} GET
