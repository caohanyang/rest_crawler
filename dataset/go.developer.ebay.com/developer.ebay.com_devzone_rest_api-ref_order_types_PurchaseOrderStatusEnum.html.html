PurchaseOrderStatusEnum

An enumeration type for the values that indicate the purchase order status.

Types that use PurchaseOrderStatusEnum:

    GuestPurchaseOrder , PurchaseOrder

Calls that use one or more values of PurchaseOrderStatusEnum:

    /guest_purchase_order/{purchaseOrderId} GET , /purchase_order/{purchaseOrderId} GET

Enumeration Values

Value Description
CANCELLED Indicates the purchase order was cancelled.
DELIVERED Indicates the purchase order has been delivered.
FULFILLMENT_IN_PROGRESS Indicates the purchase order is in the process of being shipped or is in transit.
PENDING Indicates the purchase order has submitted for payment.
* See the Enumeration Index to see exact use of each enumeration value in the API.
