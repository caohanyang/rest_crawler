CreateCampaignRequest

Type that defines the fields needed to create a campaign. To create a campaign, you need to specify a name, start and end dates, funding, and criterion (listing selection) method. There are two criterion methods; a list of inventory IDs and listing selection rules.

Call that uses CreateCampaignRequest:

    /ad_campaign POST

Fields

campaignCriterion ( CampaignCriterion )
campaignName ( string )
endDate ( string )
fundingStrategy ( FundingStrategy )
marketplaceId ( MarketplaceIdEnum )
startDate ( string )


campaignCriterion ( CampaignCriterion ) [0..?]
Type that defines the fields that specify a rule based campaign.
See the Field Index to learn which calls use campaignCriterion.

campaignName ( string ) [0..?]
The name for the campaign. You can use any alphanumeric characters except the less than or greater than characters (< >).

Max length: 80 characters

See the Field Index to learn which calls use campaignName.

endDate ( string ) [0..?]
The end date of the campaign. If it is a selection rule campaign, you can clone using the Clone Campaign call.

Valid Format: yyyy-MM-ddThh:mm.ss.sssZ

See the Field Index to learn which calls use endDate.

fundingStrategy ( FundingStrategy ) [0..?]
The funding strategy of the campaign.
See the Field Index to learn which calls use fundingStrategy.

marketplaceId ( MarketplaceIdEnum ) [0..?]
The eBay marketplace ID of the site where the campaign appears. The X-EBAY-C-MARKETPLACE-ID header values is ignored.
See the Field Index to learn which calls use marketplaceId.

startDate ( string ) [0..?]
The start date of the campaign. n this day, the service derives the keywords for each listing, creates an ad for each listing as associates the ad with the campaign and then starts the campaign. The amount of time it takes the service to start the campaign depends on the number of listings in the campaign. You can check it the campaign has started by using the Get Campaign call.

Valid Format: yyyy-MM-ddThh:mm.ss.sssZ

See the Field Index to learn which calls use startDate.
