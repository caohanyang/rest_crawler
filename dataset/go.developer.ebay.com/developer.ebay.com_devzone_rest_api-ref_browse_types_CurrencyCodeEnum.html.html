CurrencyCodeEnum

The three letter ISO 4217 code representing a world currency.

Note: For more information, see Currency codes - ISO 4217 .

Types that use CurrencyCodeEnum:

    Amount , ItemFeed

Calls that use one or more values of CurrencyCodeEnum:

    /item/{item_id} GET , /item_feed GET , /item_group/{item_group_id} GET
