Address

This type contains the details of a geographical address.

Types that use Address:

    Contact , FulfillmentStartInstruction

Calls that use Address:

    /order GET , /order/{orderId} GET

Fields

addressLine1 ( string )
addressLine2 ( string )
city ( string )
countryCode ( CountryCodeEnum )
county ( string )
postalCode ( string )
stateOrProvince ( string )


addressLine1 ( string ) [0..?]
The first line of the street address.
See the Field Index to learn which calls use addressLine1.

addressLine2 ( string ) [0..?]
The second line of the street address. This field can be used for additional address information, such as a suite or apartment number.
See the Field Index to learn which calls use addressLine2.

city ( string ) [0..?]
The city of the address.
See the Field Index to learn which calls use city.

countryCode ( CountryCodeEnum ) [0..?]
The country of the address, represented as two-letter ISO 3166-1 alpha-2 country code. For example, US represents the United States, and DE represents Germany.
See the Field Index to learn which calls use countryCode.

county ( string ) [0..?]
The county (not the country) in which the address is located. Counties typically, but not always, contain multiple cities or towns.
See the Field Index to learn which calls use county.

postalCode ( string ) [0..?]
The postal code (if any) of the address.
See the Field Index to learn which calls use postalCode.

stateOrProvince ( string ) [0..?]
The state or province within the country in which the address resides.
See the Field Index to learn which calls use stateOrProvince.
