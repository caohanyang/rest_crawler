Applicable values for /item_promotion/{promotion_id} PUTRequest.promotionStatus

DRAFT
Indicates the promotion is a draft. To schedule or make the promotion active use the Create Item Promotion call to submit the full details of the promotion.
ENDED
Indicates the promotion has ended.
INVALID
This is for internal use only for when promotion type specified in the request is invalid.
PAUSED
Indicates the promotion is currently on hold. To restart the promotion use the Resume Promotion call
RUNNING
Indicates the promotion is live on the site.
SCHEDULED
Indicates the promotion start date is in future (greater than current date). The promotion is not live yet.
