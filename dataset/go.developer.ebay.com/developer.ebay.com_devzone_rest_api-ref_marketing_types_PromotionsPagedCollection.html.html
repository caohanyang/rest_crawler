PromotionsPagedCollection

Type that defines the fields for the result set.

Call that uses PromotionsPagedCollection:

    /promotion GET

Fields

href ( string )
limit ( integer )
next ( string )
offset ( integer )
prev ( string )
promotions ( array of PromotionDetail )
total ( integer )


href ( string ) [0..?]
The URI of the current result set.
See the Field Index to learn which calls use href.

limit ( integer ) [0..?]
The limit value that was used. This sets the maximum number of promotions in a result set.
See the Field Index to learn which calls use limit.

next ( string ) [0..?]
The URI for the next result set.
See the Field Index to learn which calls use next.

offset ( integer ) [0..?]
The offset value that was used. This sets the starting position in the result set.
See the Field Index to learn which calls use offset.

prev ( string ) [0..?]
The URI for the previous result set.
See the Field Index to learn which calls use prev.

promotions ( array of PromotionDetail ) [0..?]
Container for the promotion detail fields.
See the Field Index to learn which calls use promotions.

total ( integer ) [0..?]
The total number of promotions that meet the input criteria.
See the Field Index to learn which calls use total.
