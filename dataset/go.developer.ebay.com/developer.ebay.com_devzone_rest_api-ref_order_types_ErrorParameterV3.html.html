ErrorParameterV3

An array of name/value pairs that describe details the error condition. These are useful when multiple errors are returned.

Type that uses ErrorParameterV3:

    ErrorDetailV3

Calls that use ErrorParameterV3:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId} GET , /checkout_session/{checkoutSessionId}/place_order POST , /checkout_session/{checkoutSessionId}/update_payment_info POST , /checkout_session/{checkoutSessionId}/update_quantity POST , /checkout_session/{checkoutSessionId}/update_shipping_address POST , /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId} GET , /guest_checkout_session/{checkoutSessionId}/place_order POST , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_address POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST
