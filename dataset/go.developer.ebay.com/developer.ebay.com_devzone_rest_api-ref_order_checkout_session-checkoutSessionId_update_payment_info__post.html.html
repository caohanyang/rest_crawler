Update Payment Info

POST /checkout_session/{checkoutSessionId}/update_payment_info

This call updates the payment method information of the specified eBay member checkout session.

Note: Although the X-EBAY-C-ENDUSERCTX header is not required, it is strongly recommended that you use it when submitting this call. This header specifies the risk correlation ID, which is provided by a security software library.

For example:
X-EBAY-C-ENDUSERCTX deviceId=< riskCorrelationId >
