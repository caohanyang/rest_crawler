RecipientAccountReference

Recipient account information, like PayPal email. If the payment method is PayPal, this structure contains the recipient's PayPal email address.

Type that uses RecipientAccountReference:

    PaymentMethod

Calls that use RecipientAccountReference:

    /payment_policy POST , /payment_policy/get_by_policy_name GET , /payment_policy/{payment_policy_id} PUT , /payment_policy/{paymentPolicyId} GET

Fields

referenceId ( string ) [0..?]
Contains the PayPal email address of the recipient (buyer) if referenceType is set to PAYPAL_EMAIL .
See the Field Index to learn which calls use referenceId.

referenceType ( RecipientAccountReferenceTypeEnum ) [0..?]
A reference a recipient's account. Currently only PAYPAL_EMAIL is valid.
See the Field Index to learn which calls use referenceType.
