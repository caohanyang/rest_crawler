TransactionVolume

A view of a merchant's transaction volume, based on a specified lookback period.

No calls use TransactionVolume.

Fields

firstOrderDate ( string ) [0..?]
Not used by any call.

Contains the time of the first transaction that is returned in the transaction snapshots, specified in DateTime format.
See the Field Index to learn which calls use firstOrderDate.

transactionSnapshots ( TransactionSnapshot ) [0..?]
Not used by any call.

A list of the merchant's transaction snapshots (histories).
See the Field Index to learn which calls use transactionSnapshots.
