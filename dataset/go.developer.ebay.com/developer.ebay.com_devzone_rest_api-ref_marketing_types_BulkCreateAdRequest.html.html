BulkCreateAdRequest

Type that defines the fields for the create ads in bulk request.

Calls that use BulkCreateAdRequest:

    /ad_campaign/{campaign_id}/bulk_create_ads_by_listing_id POST , /ad_campaign/{campaign_id}/bulk_update_ads_bid_by_listing_id POST

Field

requests ( array of CreateAdRequest ) [0..?]
An array of the eBay listing IDs and the bid percentage for each to be used to create ads.

Maximum IDs per call: 500

See the Field Index to learn which calls use requests.
