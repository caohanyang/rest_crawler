OrderPaymentStatusEnum

The current status of combined monetary exchanges (payment to seller and refund to buyer) for an order.

Type that uses OrderPaymentStatusEnum:

    Order

Calls that use one or more values of OrderPaymentStatusEnum:

    /order GET , /order/{orderId} GET

Enumeration Values

Value Description
FAILED One or more monetary exchanges has failed.
FULLY_REFUNDED Complete — the entire order has been refunded.
PAID Complete — the order is fully paid.
PARTIALLY_REFUNDED Complete — a portion of the order has been refunded.
PENDING The monetary exchanges are on hold.
