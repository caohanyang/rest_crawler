Supported Locales

Following are the locales Yelp Fusion API supports. The locale code is in the format of {language code}_{country code}, where:

  • Language code mostly follows ISO 639-1 alpha-2 code. When such a code is unavailable for a language, it will be a ISO 639-2 alpha-3 code instead.
  • Country code follows ISO 3166-1 alpha-2 code.
